# Coin Animation System

## DOTween Installation Instructions

To use the coin animation system, you need to install DOTween in your Unity project:

1. Open Unity Package Manager (Window â†’ Package Manager)
2. Click the "+" button and select "Add package from git URL..."
3. Enter the following URL: `https://github.com/Demigiant/dotween.git`
4. Click "Add" to install the package

Alternatively, you can manually download DOTween from the Asset Store:
1. Open Unity Asset Store
2. Search for "DOTween"
3. Download and import the package

After installation, the system will automatically initialize DOTween when needed.
